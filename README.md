# My GitHub Index

自分の GitHub リポジトリを一覧表示・検索できるインデックスサイト。

## 概要

GitHub には自分のリポジトリ全体を俯瞰できる「インデックス」がない。そのため、GitHub の中にどこに何があるか把握しづらい。
本プロジェクトは GitHub の HTML（GitHub Pages 等）機能を活用し、「まずはここを見に行けば良い」という入り口となるインデックスサイトを提供する。
